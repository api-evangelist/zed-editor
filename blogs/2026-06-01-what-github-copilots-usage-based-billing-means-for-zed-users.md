GitHub is changing Copilot Chat from request-based billing to token-based billing, meaning Zed users leveraging Copilot Chat will see their consumption tracked through GitHub AI Credits rather than the previous Premium Request Unit system. Code completions and Next Edit suggestions remain unaffected by this modification. The post explains what the billing change means practically for Zed users and how to monitor consumption going forward.
